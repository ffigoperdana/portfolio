## Overview

Calendar MSI is an event-management platform for PT. Mastersystem Infotama Tbk. It brings calendar activity, custom booking forms, and data export into one workflow instead of relying on a calendar event alone.

## Problem

Teams needed a structured way to capture event details and keep operational data accessible alongside the calendar schedule.

## Approach

Built the client with React and JavaScript, integrating Google Calendar and Firebase/Google configuration. The service is containerized with a multi-stage Docker build, served through Nginx, and verified by a GitHub Actions pipeline that tests, builds, and publishes the container image.

## Results

Project records report a 25% increase in user engagement and a 35% improvement in data accessibility.
