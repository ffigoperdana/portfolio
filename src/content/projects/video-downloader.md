## Overview

Video Downloader is a self-hosted image and video downloader available at save.fgdev.tech. It is built with TypeScript across the application and packaged for deployment with Docker.

## Approach

The project uses a single TypeScript-based application stack and containerized deployment, keeping the local development and hosted environments aligned.
