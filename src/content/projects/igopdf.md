## Overview

igoPDF is an internal document-processing platform customized and deployed for BPDP. It keeps PDF processing on the client side, reducing the need to send organizational documents to third-party services.

## Approach

The platform runs on TypeScript and Node.js/Express with PostgreSQL, Docker, and Nginx. It adds internal authentication, Active Directory/LDAP integration, session security, CAPTCHA, role-based access control, automatic user provisioning, and administrative user management.

## Operations

Usage reporting and operational configuration support day-to-day administration while the privacy-first processing model reduces document exposure.
