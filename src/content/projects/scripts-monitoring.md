## Overview

Scripts Monitoring is a collection of shell-based utilities for monitoring target virtual machines and supporting routine infrastructure operations.

## Approach

The tooling stays close to the operator environment: lightweight Bash scripts can be run and adapted without a separate application stack. The repository keeps the scripts versioned for repeatable operational use.
