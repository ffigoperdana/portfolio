## Overview

MyGram is a social-media REST API built with Go, Gin, PostgreSQL, and GORM. It provides the backend foundation for user, photo, comment, and social-link workflows.

## Approach

JWT authentication, authorization rules, request validation, and environment-based configuration support a secure service boundary. Docker keeps development and deployment setup consistent.
